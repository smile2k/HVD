## 1.1  ?=
```CSharp
variable ??= value;
// <=>
if (variable == null)
{
    variable = value;
}
```
Toán tử `??=` trong C# là toán tử gán hợp nhất null (null-coalescing assignment operator). Nó được sử dụng để gán một giá trị cho biến chỉ khi biến đó hiện đang có giá trị là `null`.
